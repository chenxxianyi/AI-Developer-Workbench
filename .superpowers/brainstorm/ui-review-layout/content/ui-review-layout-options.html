A. 聚焦任务流（推荐）

保留双栏，但强化页面唯一 H1、主 CTA、模式说明、空状态和焦点态。改动集中，风险低，最贴合报告。

B. 分步向导

把输入、上传、结果拆成步骤。新手友好，但会增加交互复杂度，也可能降低熟练用户效率。

C. 深色开发者工作台

更酷、更有产品记忆点，但会牵动全局主题、对比度和更多页面，适合作为后续大版本。

我建议先做 A 的具体范围

去掉重复返回/标题干扰；增强主 CTA；补模式说明；改善空状态；统一 focus-visible、浅灰对比和上传区域可操作感；保持响应式单栏兜底。
