# A-08 MVP 范围清单

> 第一版本（MVP）功能范围，后续功能进入变更流程。

## ✅ MVP 包含

### 认证
- [ ] 用户注册（邮箱 + 密码）
- [ ] 用户登录（JWT Token）
- [ ] Token 刷新和过期处理
- [ ] 个人资料修改
- [ ] 密码修改

### Dashboard
- [ ] 项目总数统计
- [ ] 生成中/已完成/失败项目统计
- [ ] 最近项目和最近任务列表
- [ ] 空状态、加载状态、错误状态

### 项目
- [ ] 项目列表（分页、搜索、筛选）
- [ ] 创建项目向导（生成网站 / 分析已有项目）
- [ ] 项目详情（状态、质量分）
- [ ] 项目编辑
- [ ] 项目软删除

### 生成流程
- [ ] 需求表单（目标、用户、功能、页面、风格、技术栈）
- [ ] 需求保存和恢复
- [ ] 蓝图生成（AI）
- [ ] 蓝图查看（产品定位、页面、组件、数据库、API、UI、技术栈）
- [ ] 蓝图重新生成
- [ ] 蓝图确认
- [ ] 代码生成 Pipeline
- [ ] SSE 实时进度推送
- [ ] 生成失败重试
- [ ] 任务取消

### 构建与预览
- [ ] 项目构建（npm install + build）
- [ ] 在线预览（iframe + 预览会话）
- [ ] 预览刷新和重新构建
- [ ] 新窗口打开

### 文件与导出
- [ ] 项目文件树浏览
- [ ] 文件内容查看
- [ ] ZIP 导出（排除敏感文件）

### AI 开发工具
- [ ] UI Review（支持项目上下文）
- [ ] Project Doctor（支持项目上下文）
- [ ] Agent Config
- [ ] API Doc
- [ ] DB Schema

### 报告
- [ ] 报告列表（按项目筛选）
- [ ] 报告详情
- [ ] 报告对比
- [ ] 报告关联项目

### 管理后台
- [ ] AI 模型管理（CRUD、启停、密钥加密）
- [ ] Prompt 模板管理（CRUD、启停）
- [ ] 用户管理（列表、状态管理）
- [ ] 项目管理（列表、查看）

## 范围变更流程

MVP 范围冻结后，新增需求进入以下流程：
1. 提出变更申请（说明价值、成本、风险）
2. 评审（是否阻塞 MVP、是否有替代方案）
3. 批准后列入对应版本计划，不直接插入当前里程碑
